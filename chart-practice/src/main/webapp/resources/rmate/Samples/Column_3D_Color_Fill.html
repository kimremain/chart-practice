컬럼 차트를 단일 색상으로 변경하는 샘플입니다.
  • Series안에 색을 채울 때(단일색) fill에 주목
  • color : 색상
  • alpha : 투명도
<Column3DSeries yField="Profit">
	<fill>
		<SolidColor color="#FF5050" alpha="1"/>
	</fill>
	<stroke>
		<Stroke weight="1" color="#FF5050"/>
	</stroke>
	<showDataEffect>
		<SeriesInterpolate/>
	</showDataEffect>
</Column3DSeries>
