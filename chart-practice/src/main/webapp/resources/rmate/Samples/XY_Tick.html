축의 스타일을 변경하는 샘플입니다.
  • visible : 축 자체를 보이기 안보이기
  • tickLength : tick의 길이
  • minorTickLength : 축에 표시되는 tick의 길이
  • tickPlacement - cross, inside, outside : tick이 그려지는 방향
  • placement - bottom,left,right,top : 축이 표시될 방향
  • showLabels : Label보이기 안보이기
  • labelAlign - left, center, right : Label의 표시지점
<Axis3DRenderer axis="{hAxis}" tickLength="10" minorTickLength="0" 
tickPlacement="outside" minorTickPlacement="cross" placement="bottom" 
canDropLabels="true" showLabels="true"
showLine="true" labelAlign="center">
	<axisStroke>
		<Stroke weight="10" color="#4C9CFF" caps="none"/>
	</axisStroke>
	<tickStroke>
		<Stroke weight="1" color="#00549b" alpha="1"/>
	</tickStroke>
	<minorTickStroke>
		<Stroke weight="1" color="#FFFFFF" caps="square"/>
	</minorTickStroke>
</Axis3DRenderer>
