차트의 배경에 이미지를 적용하는 샘플입니다.
  • Image
  • source : 실제 이미지가 있는 URL경로입니다
  • maintainAspectRation : 이미지 원본 비율을 유지할지를 나타냅니다 True면 고유비율을 유지하려 하고 False면 차트 크기에 맞춘 사이즈를 갖습니다
  • alpha : 이미지의 투명도를 나타 냅니다.
<backgroundElements>
	<Image source="http://www.riamore.net/image/riamoreMain.jpg" 
	maintainAspectRatio="false" alpha="1"/>
</backgroundElements>
