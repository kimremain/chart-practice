X축 라벨을 사용자가 정의하여 사용하는 샘플입니다.
  • 스크롤 차트를 생성했을 경우 축 라벨을 사용자 정의 합니다.
  • layout XML 에서 축 렌더러 속성을 넣을때 labelJsFunction 주고, 만든 javascript 함수명을 넣어줍니다.
 <horizontalAxisRenderers>
	<ScrollableAxisRenderer axis="{hAxis}" visibleItemSize="7" 
	labelJsFunction="axisLabelFunc" fontFamily="맑은 고딕"/>
</horizontalAxisRenderers>


//파라메터 설명
//id : 축의 ID
//value : 현재 아이템에 맞는 축의 라벨 값
function axisLabelFunc(id, value)
{
	return  value + ", 서울시";
}
