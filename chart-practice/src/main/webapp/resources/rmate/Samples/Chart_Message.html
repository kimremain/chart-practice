• Invalid License : 정상적인 라이센스 키가 아닐 경우
  • This host is not autohrized : 차트가 출력된 host가 라이센스 발급 시 요청한 host가 아닐 경우
  • Period has expired : 라이센스 키 만료일자가 지났을 경우
  • Other versions of the license : 다른 제품의 라이센스 키를 설정하고 있을 경우
  • Evaluation license : 제품이 정품인데 평가판 라이센스 키를 설정하고 있는 경우
  • Trial license is required : 제품이 평가판인데 라이센스 키가 평가판이 아닐 경우
  • Invalid LicenseUrl : LicenseUrl이 올바르지 않을 경우
