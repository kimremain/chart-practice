파이 차트 안쪽에 수치를 표시하는 샘플입니다.
  • labelPosition을 inside으로 설정 하였습니다.
  • none : 데이터 값을 표시 하지 않습니다
  • inside : 차트 안에 표시합니다
  • outside : 차트 밖에 표시합니다
  • callout : 차트 밖에 따로 선을 긋고 표시합니다
  • insideWithCallout : 기본적으로 안쪽에 표시하지만 크기가 작아 공간이 없을 때는 선을 따로 만들어 긋고 표시합니다
  • <Pie2DSeries field="Profit" nameField="Month" displayName="Profit" labelPosition="inside" color="#ffffff" />
